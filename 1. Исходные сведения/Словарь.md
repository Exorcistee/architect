Ключевые термины, которые полезно объяснить.

1. **Клиент (Пользователь)** - Физическое лицо, зарегистрированное на платформе с целью поиска и получения консультаций у Экспертов.
2. **Эксперт (Консультант)** - Физическое или юридическое лицо, прошедшее верификацию и предоставляющее платные консультационные услуги на платформе. Имеет профиль с рейтингом.
3. **Сеанс (Консультация)** - Ограниченный по времени сеанс взаимодействия между Клиентом и Экспертом, проводимый через один из доступных каналов связи (видео, аудио, текст). Является единицей биллинга.
4. **Канал связи**	- Технологический способ проведения консультации: Видео-чат, Аудио-чат, Текстовый чат.
5. **Бронирование** - Договоренность между Клиентом и Экспертом о проведении Сеанса в определенное время, подтвержденная созданием записи в системе. Может быть бесплатной, условно-оплаченной (холд средств) или оплаченной.
6. **Расписание** - Электронное расписание Эксперта, отражающее его доступность для бронирования Сеансов. Управляется Экспертом.
7. **Тариф** - Стоимость, установленная Экспертом за единицу времени (минута/час) или за тип Сеанса. Может различаться для разных Каналов связи.
8. **Событие домена** - Неизменяемая запись о факте, который произошел в предметной области (например, SessionStarted, PaymentSucceeded, ReviewSubmitted). Используется для асинхронной коммуникации между микросервисами в рамках Event-Driven архитектуры.
9. **Сквозное шифрование (E2EE)** -	Метод шифрования, при котором данные (текст сообщений, медиапотоки) шифруются на устройстве отправителя и расшифровываются только на устройстве получателя, будучи недоступными для расшифровки серверами платформы.
10. **WebRTC (Web Real-Time Communication)** - Набор протоколов и API для организации peer-to-peer коммуникаций в реальном времени прямо в браузере или мобильном приложении. Основа для Сервиса видеосвязи.
11. **PCI DSS (Payment Card Industry Data Security Standard)**	- Мировой стандарт безопасности данных индустрии платежных карт. Требует строгого контроля над обработкой, хранением и передачей данных карт.
